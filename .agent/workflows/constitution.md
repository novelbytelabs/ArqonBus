---
description: Update the project constitution. Usage: /speckit.constitution
---

1. Load `.specify/templates/constitution-template.md` (if available) or the current authoritative constitution at `.specify/memory/constitution.md`.
2. Analyze the user's request for amendments or syncing.
3. If responding to a specific change request:
   - Modify the Constitution text.
   - Update the "Last Amended" date.
   - Verify alignment with existing core principles (Voltron, Tier Omega).
4. Save the updated Constitution to `.specify/memory/constitution.md`.
5. Notify the user of ratification.
